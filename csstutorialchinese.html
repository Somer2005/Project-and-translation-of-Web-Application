CSS 学习记录与速查表

这篇文档将作为我学习 CSS 的记录， 和 HTML 那一篇一样，它的目标并不是系统教学， 而是构建一个“用得多、查得快”的速查表。

在我目前的理解里，CSS 并不是一门“难”的东西， 但它非常零散，规则多、例外多， 如果只是看教程而不反复使用， 很容易陷入“看过但不会写”的状态。

因此，这份文档会优先记录： 常见属性、常见写法、常见坑， 而不是追求覆盖所有规范细节。

第零部分 CSS 是用来干什么的

在正式开始写 CSS 之前， 有必要先明确一件事情：CSS 并不负责“内容是什么”， 而是负责“内容该怎么显示”。

HTML 回答的是“这是什么”， 而 CSS 回答的是“它看起来应该是什么样子”。

在浏览器中， HTML 构建的是文档结构， CSS 则是在这套结构之上， 定义颜色、大小、布局和视觉层级。

第一部分 CSS 的三种引入方式

CSS 并不是只能写在一个地方。 根据使用场景的不同， 它可以通过三种方式引入到 HTML 文档中。

1. 内联样式（inline style）

内联样式是指直接写在 HTML 标签上的 CSS,哎呀其实说白了ius直接在HTML的块上设置一个。 它的特点是作用范围最小、优先级最高， 但可维护性极差。

当然说是什么说，还是那句话，又不是造火箭……


<p style="color: red; font-size: 16px;">
    这是一段使用内联样式的文字
</p>

2. 内部样式表（style 标签）

内部样式表是将 CSS 写在 HTML 文件的 <style> 标签中， 通常放在 <head> 内部。 


<style>
    p {
        color: blue;
        font-size: 16px;
    }
</style>

这种方式适合页面结构简单、 不需要复用样式的场景， 但仍然不利于多页面维护。

PS:我觉得这是最垃圾的一种

3. 外部样式表（推荐）

外部样式表是将 CSS 写在独立的 .css 文件中， 再通过 <link> 标签引入。 这是最常见、也是最推荐的做法。 


<link rel="stylesheet" href="stylecss.css">

这种方式可以实现样式复用， 也能让 HTML 和 CSS 各自保持清晰的职责边界。

在后续内容中， 除非特别说明， 本文档默认使用外部样式表的方式编写 CSS。

第二部分 常见 CSS 设置项及其作用

在实际写 CSS 的过程中， 真正高频使用的属性并不多。 大多数时候，我们是在反复使用同一批属性， 只是组合方式不同。

这一部分将记录一些最常见、最基础， 也是最容易在初学阶段产生混淆的 CSS 设置项。

1. display：元素如何参与布局

display 属性决定了一个元素在页面中的“基本行为方式”， 也就是它是独占一行，还是和其他元素并排， 以及它是否参与布局计算。

最常见的几种取值如下。 


display: block;
display: inline;
display: inline-block;
display: none;

block 元素会独占一整行， 常见的如 div、p、h1 等； inline 元素不会换行， 宽高通常由内容决定； inline-block 则介于两者之间， 既可以设置宽高，又可以并排显示。

display: none 表示元素完全不参与页面布局， 它既不会显示，也不会占据空间。

2. flex：一种布局方式，而不是单个属性

flex 并不是一个独立的属性， 而是一整套布局模型。 当一个元素被设置为 flex 容器时， 它的直接子元素将按照“弹性布局”的规则进行排列。 


.container {
    display: flex;
}

一旦启用 flex， 布局的关注点就从“元素本身”转移到了“父子关系”上。 flex 容器控制的是子元素如何排列， 而不是子元素自己决定位置。

最常见的几个 flex 相关属性包括： 


flex-direction: row;
justify-content: center;
align-items: center;

flex-direction 决定主轴方向； justify-content 控制子元素在主轴上的排列方式； align-items 控制子元素在交叉轴上的对齐方式。

在很多居中布局场景下， flex 是目前最直接、最稳定的解决方案。

3. width 与 height：尺寸的直接控制

width 和 height 用于设置元素的宽度和高度， 但它们是否生效， 取决于元素的 display 类型。 


.box {
    width: 200px;
    height: 100px;
}

对于 inline 元素， width 和 height 通常是无效的； 而 block 和 inline-block 元素， 则可以正常设置尺寸。

4. margin 与 padding：元素之间与元素内部的空间

margin 用于控制元素与元素之间的距离， padding 用于控制元素内容与边框之间的距离。 


.box {
    margin: 20px;
    padding: 10px;
}

这两个属性并不会改变内容本身， 但会直接影响页面的整体布局和视觉节奏。

5. border：元素的边界

border 用于给元素添加边框， 它由宽度、样式和颜色三部分组成。 


.box {
    border: 1px solid #000;
}

在调试布局时， 给元素临时加上 border， 是一种非常常见也非常有效的手段。

6. color 与 background-color：文字与背景

color 用于设置文字颜色， background-color 用于设置元素的背景色。 


.text {
    color: #333;
    background-color: #f5f5f5;
}

需要注意的是， background-color 作用于整个元素区域， 包括 padding 和内容区域。

以上这些属性， 构成了日常写 CSS 时最基本的一套工具。 后续更复杂的布局和效果， 本质上都是在这些基础属性之上的组合与扩展。
